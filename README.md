# sticker2gif

A tool to download Facebook Messenger Stickers and convert them into gif using basic AI.


## Dependencies

`pip install Pillow`

## Usage

```python
from sticker2gif import Maker

url = input('URL: ')

tool = Maker(url, log=True)

tool.run()
```
